# Fibonacci

It can run to Fib(934) and verify the number is correct.

## Running the fib program

Compile you code to create the testing program `fib`
```shell
$ make
```

Check the correctness of fib
```shell
$ make check
```

## Using fib

`fib` is a fibonacci number program and this is an example:

```shell
$ ./fib 10
```
